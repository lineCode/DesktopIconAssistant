桌面图标排列助手(Desktop Icon Assistant)
=======

![preview](http://yxl.github.com/DesktopIconAssistant/images/preview.png)

你想在桌面上书写自己的“爱”吗？ 请拿起这个小具，把你的桌面图标排成心形，为生活添一份爱。

桌面图标排列助手还能实现图标靠右，靠上和靠下排列。

它还可以将桌面图标排形成指定大小和位置的直线、曲线、矩形、椭圆和心形，可排列部分图标。

这个工具的操作方法和windows自带的“画图”类似，你可以随心所欲地“绘制”自己的桌面，让图标一笔一笔地排成你想要的图形。

基本功能　
=========

1.保存和恢复桌面图标位置。

2.将选定的桌面图标排列成直线、曲线、矩形、椭圆或心形。

3.隐藏桌面图标文字。(只对WinXP有效)
